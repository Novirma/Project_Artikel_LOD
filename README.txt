1. Ini adalah Sebuah Aplikasi CRUD Article
2. Untuk BackEnd nya menggunakan Code Igniter 3 yang di runing menggunakan XAMPP V3.3.0
3. Untuk FrontEnd nya Menggunakan VueJS 3
4. Untuk Databasenya Menggunakan PhpMyAdmin
5. Sebelum Menjalankan Aplikasi Importlah Tabel Database dari file "db_artikel.sql" dan di tampung di database yang di beri nama "project_lod" agar tidak terjadi error dan juga di tampung dalam sebuah tabel yang bernama "db_artikel"
6. Jangan lupa juga untuk mengimport API Postman jika ingin menguji aplikasi lewat server BackEnd nya
7. Jangan lupa untuk meletakkan Aplikasi Back End ke dalam folder htdocs/project_lod karena di jalankan menggunakan Aplikasi XAMPP,jika tidak ada folder "project_lod" maka di harapkan agar membuat folder tersebut dan meletakkannya di sana
8. Jangan lupa untuk melakukan penginstalan NodeModules pada aplikasi Front End VueJS 3 dengan "npm install"
9. VueJS dapat di runing di terminal dengan perintah "npm run dev"
